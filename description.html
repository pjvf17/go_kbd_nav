A forked version of ogs-kbd-nav. Updated to work properly with Firefox, and added configurable shortcuts. Main description here: go_kbd_nav Keyboard navigation for OGS, the best online platform for playing the game of. Now, instead of only being able to play the ancient game online with your mouse, you can do it with your keyboard as well. Other handy shortcuts are also included.
  • Ctrl + \: Global switch: enables or disables the whole extension
  • Ctrl + m: Toggle the chat input box on a game page
  • Ctrl + b: Toggle the canvas overlay with the stone marker
  • Ctrl + ]: Toggle the arrow keys
  • w or ↑: Move the stone marker up
  • a or ←: Move the stone marker left
  • s or ↓: Move the stone marker down
  • d or →: Move the stone marker right
  • Enter: Click on the stone marker's location
  • Ctrl + [: Pass
  • Ctrl + ,: Cycle through the different board sizes
  • Ctrl + .: Toggle the coordinates input field
  • j: Press the submit button (correspondence)
  • Shift + e: Press the submit button, even in coordinates input
  • Escape: exit coordinates input or chat (not configurable)
To configure:
  1. Go to about:addons
  2. Scroll down to Go Keyboard Navigation and click on it
  3. Click on preferences, follow instructions
