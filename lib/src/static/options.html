Press Record Shortcut to overwrite the current shortcut for a specific action.
Press Reset to restore the default shortcuts.
If you're logged in to your browser your shortcuts will sync with that account accross devices. After you change the shortcuts, reload any ogs tab you have open in order to use the new keybinds

A shortcut can be any combination of modifer keys + a normal key, ex: (shift+ctrl+e). Or a sequence of key presses, like (g i). Be aware that if you have something bound to the first key of a sequence, it won't trigger that action because it's listening for the next key
